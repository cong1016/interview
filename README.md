使用django建立一个玩家分数排行榜服务，服务向客户端提供两个接口:

客户端上传客户端号和分数(注意：并不会上传排名,客户端无法上传排名),同一个客户端可以多次上传分数，取最新的一次分数
客户端查询排行榜


例子说明


10个客户端分别调用一次接口1，分数范围为1...10000000

可以查询任何名次段，例如可以查询排名20~30的表格
